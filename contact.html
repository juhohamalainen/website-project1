Returns by post


Complete the returns form by entering the return quantity and return code for each item. Enclose the completed dispatch note in the parcel along with the item(s) in the original presentation box, and package up securely. Both a pre-paid Royal Mail barcode and a pre-addressed Watcher returns label are included in your delivery parcel. Peel and affix the Fraser Hart label to the front of the package and affix the Royal Mail to the top right-hand side of the box. Please remember to obtain proof of posting from the Post Office. There is no charge for the return as this is pre-paid. Please allow 30 days from us receiving the return goods to process your refund.
